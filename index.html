The assigment

For assignment 4 of the Semantic Web course a website needed to be created. This website should be able to send a SPARQL query to a Stardog triple store. The database used for this assignment is the database created with assignment 3. In this case the database is about hamburgers. The assigment consists of two parts: in the first part the query will not make use of inferencing and in the second part of the assignment the query will make use of inferencing. Inferencing means that, based on the data and some set of rules, new relationships can automatically be generated.
